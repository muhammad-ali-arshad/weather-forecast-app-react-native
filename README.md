# Weather Forecast App (MERN + React Native)

A full-stack weather forecast application built using:

- React Native (Mobile App)
- Node.js & Express (Backend)
- MongoDB (Database)
- Weather API (Real-time data)

## Features
- Real-time weather updates
- City-based search
- Clean & modern UI
- REST API backend

## Tech Stack
- MERN Stack
- React Native
- OpenWeather API
